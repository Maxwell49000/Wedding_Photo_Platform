# Wedding Photo Platform

Private collaborative photo gallery for wedding guests.

This repository is being scaffolded from the project specification in
`Wedding_Photo_Platform_Master_Specification.md`.

## Target architecture

- Frontend: Vue 3, Composition API, Vuetify, Pinia, Axios, Vue Router
- Backend: Java 21, Spring Boot, Spring Data JPA, Validation, Lombok
- Database: PostgreSQL
- Infra: Docker, Docker Compose, Nginx

## Current status

Initial repository structure is being prepared. The next steps are to
add the frontend and backend application skeletons, then wire them into
Docker.

## Environment

Copy `.env.example` to `.env` if you want to override the defaults used
by `docker-compose.yml`.
